# Bibbidi Brownie - Website

Site vitrine da Bibbidi Brownie, empresa de doces artesanais.

## Tecnologias

- React 19
- Vite
- Tailwind CSS 4
- TypeScript

## Deploy

Este projeto está configurado para deploy automático no Vercel.
